## Quick Start

### 1. Setup Environment
```bash
cd /home/get/Desktop/indodax-trading-bot
python -m venv venv
source venv/bin/activate  # Linux/Mac
pip install -r requirements.txt
```

### 2. Configure
```bash
cp .env.example .env
nano .env  # Isi API keys
python utils/encrypt_keys.py  # Encrypt API keys (opsional)
```

### 3. Run
```bash
# Backtest
python main.py backtest --pairs BTC_IDR --timeframe 1h --start 2023-01-01 --end 2023-12-31

# Paper Trading (Simulasi)
python main.py paper --pairs BTC_IDR,ETH_IDR --timeframe 1h

# Live Trading (REAL MONEY - HATI-HATI!)
python main.py live --pairs BTC_IDR --timeframe 1h
```
